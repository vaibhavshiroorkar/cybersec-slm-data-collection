# Risk Register

## Week 2 Risks

| Risk | Likelihood | Impact | Mitigation |
|------|------------|--------|------------|
| API rate limits hit during extraction | High | Medium | Implement retry logic and crawl delays |
| Source blocks scraper | Medium | High | Rotate user agents, respect robots.txt |
| License status unclear on yellow sources | Medium | High | Flag and seek clarification before use |
| Data volume lower than estimated | Medium | Medium | Identify backup sources in Week 1 |
| Storage runs out mid-extraction | Low | High | Provision extra headroom before Week 2 |
| Team member unavailable | Low | High | Document all scripts so anyone can run them |
