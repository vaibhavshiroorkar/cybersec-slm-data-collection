# Source Selection Criteria

A source is worth pursuing if it meets most of these:

- Relevance: contains cybersecurity text (CVEs, malware, threats, research)
- Size: at least a few thousand documents or records
- Language: primarily English
- Access: publicly accessible without heavy restrictions
- License: permits use for ML training (green or yellow)
- Update frequency: actively maintained or large enough as a static dump
- Extraction difficulty: realistically scriptable within the project timeline
